Route Generator

This page makes a single call to the Google Directions API to generate the route from HQ to Bucharest. The result is saved as route.json — upload it to GitHub and the map pages will use it forever, with no further API calls.

  2. Commit and push to GitHub
  3. That's it — the map pages will now use this file instead of calling the API
